# Bus Tracker using Prasarana's API

## Provided by [data.gov](https://developer.data.gov.my/realtime-api/)

### Setup

1. Clone the repository
2. Install the required packages

```bash
bun i
```

3. Add the .env file

```bash
MONGODB_URI=your_mongodb_uri
```

4. Run the server

```bash
bun dev
```
